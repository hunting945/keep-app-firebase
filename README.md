### 下載
```
git clone https://github.com/hunting945/keep-app-firebase
```
### 安裝
```
npm install
```
### 加入Firebase Init Config Data
```
var firebaseConfig = {
    apiKey: "",
    authDomain: "",
    databaseURL: "",
    projectId: "",
    storageBucket: "",
    messagingSenderId: "",
    appId: ""
};
```

### 啟動
```
npm start
```
